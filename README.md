## 🛠 About core

core is a Go-based tool that helps you manage users, permissions, and access rights. It does not rely on any specific system, so it works well with many setups. It supports modern features like secure tokens (JWT), multiple user groups (multi-tenant), and role-based access control (RBAC). You can use it with common databases like MySQL, PostgreSQL, MongoDB, and Redis.

This tool helps organizations control who can do what inside their apps or websites. It also works behind the scenes to keep data safe and organized.

- Admin: Full control
- Manager: Limited control over some features
